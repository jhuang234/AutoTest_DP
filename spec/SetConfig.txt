Select the connector type of the DUT, either 'Standard DP/mDP' or 'USB Type-C'. Note: The connector type will affects
the cable model used for TP3 (TP3_CTLE / TP3_EQ) tests.

ARSL CommandLine Utility:  arsl -a <app's ip address> -c "<command>"

----- IRemoteAte Methods -----
GetConfig 'ConnectorType'

SetConfig 'ConnectorType' '<value>'

----- Parameters -----
<value> = One of the following:
    USB Type-C
    Enhanced FsDP
    Enhanced mDP

----- Run-time Parameter Discovery -----
GetAllOptionsForMember 'ConfigValues' 'ConnectorType'

----- Note -----
Select the connector type of the DUT, either 'Standard DP/mDP' or 'USB Type-C'. Note: The connector type will affects
the cable model used for TP3 (TP3_CTLE / TP3_EQ) tests.
For more information, please see the N5452A Remote Programming Toolkit documentation (www.keysight.com/find/rpi).

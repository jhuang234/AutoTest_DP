Enable or disable predefined optimum Preset Number for TP2 tests.

ARSL CommandLine Utility:  arsl -a <app's ip address> -c "<command>"

----- IRemoteAte Methods -----
GetConfig 'PredefinedTP2PresetNumber'

SetConfig 'PredefinedTP2PresetNumber' '<value>'

----- Parameters -----
<value> = One of the following:
    0.0
    1.0

----- Run-time Parameter Discovery -----
GetAllOptionsForMember 'ConfigValues' 'PredefinedTP2PresetNumber'

----- Note -----
Enable or disable predefined optimum Preset Number for TP2 tests.
For more information, please see the N5452A Remote Programming Toolkit documentation (www.keysight.com/find/rpi).

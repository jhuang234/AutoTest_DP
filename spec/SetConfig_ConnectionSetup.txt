Select the connection type between DUT and Scope. Either 'SingleEnded'

ARSL CommandLine Utility:  arsl -a <app's ip address> -c "<command>"

----- IRemoteAte Methods -----
GetConfig 'ConnectionType'

SetConfig 'ConnectionType' '<value>'

----- Parameters -----
<value> = One of the following:
    Differential
    Single Ended
    Switch Matrix
    Common

----- Run-time Parameter Discovery -----
GetAllOptionsForMember 'ConfigValues' 'ConnectionType'

----- Note -----
Select the connection type between DUT and Scope. Either 'SingleEnded'
For more information, please see the N5452A Remote Programming Toolkit documentation (www.keysight.com/find/rpi).

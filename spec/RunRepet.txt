The repetition mode to use for the next run.

ARSL CommandLine Utility:  arsl -a <app's ip address> -c "<command>"

----- IRemoteAte Properties -----
RunRepetition?

RunRepetition '<mode>'

----- Parameters -----
<mode> = One of the following:
    Once
    N Times
    Forever

----- Run-time Parameter Discovery -----
GetAllOptions 'RunRepetitionNames'
GetCurrentOptions 'RunRepetitionNames'

For more information, please see the N5452A Remote Programming Toolkit documentation (www.keysight.com/find/rpi).
